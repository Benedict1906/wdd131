Overview

Purpose

This website is a personal photography portfolio that highlights my work, offers booking options, and provides insight into my creative process. It will serve both as a professional presence and an interactive experience for visitors.

Audience

The audience includes photography enthusiasts, clients looking to hire a photographer for events or portraits, and anyone interested in visual storytelling and creative inspiration.

Dynamic Elements

JavaScript will power an interactive photo gallery with filtering options, a dynamic hero image that changes daily, and modals for viewing images in full screen. Additional elements will include navigation toggling and user feedback based on gallery interaction.

Content

Home Page

The homepage will include a hero image, welcome message, short photographer bio, featured image of the day, and links to view the gallery or contact me.

Gallery Page

The gallery page will include a categorized image grid (Portraits, Landscapes, Events) with JavaScript-driven filtering. Clicking on a thumbnail opens a modal view. Each image will have a caption and optional metadata.
